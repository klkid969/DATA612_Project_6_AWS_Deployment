# DATA 612 – Project 6: AWS Deployment

## Project Overview

This project demonstrates how a Spark-based movie recommender system can be deployed using Amazon Web Services (AWS). Rather than building a new recommender system, this project extends the work completed in Project 5 by creating a cloud environment capable of supporting the application.

The deployment uses AWS Free Tier eligible services to demonstrate the relationship between cloud storage, cloud computing, and network security while providing hands-on experience with cloud technologies.

---

## Objectives

- Deploy a Spark-based recommender system using AWS.
- Store the MovieLens 1M dataset in Amazon S3.
- Launch an Amazon EC2 virtual machine.
- Understand the role of Amazon VPC in cloud networking.
- Evaluate the advantages and limitations of using cloud computing for recommender systems.

---

## AWS Services Used

- **Amazon S3** – Cloud storage for the MovieLens dataset.
- **Amazon EC2** – Virtual machine for cloud computing resources.
- **Amazon VPC** – Secure networking environment connecting AWS resources.

---

## Dataset

**MovieLens 1M Dataset**

- Approximately 1 million movie ratings.
- Source: GroupLens Research, University of Minnesota.

---

## Project Highlights

- Created an AWS Free Tier account.
- Configured Amazon S3 cloud storage.
- Uploaded the MovieLens dataset.
- Created and launched an Amazon EC2 instance.
- Explored Amazon VPC networking.
- Documented the cloud deployment process with screenshots.

## What I Learned

This project provided my first meaningful hands-on experience using AWS. Although I previously created an AWS account while completing my A.A.S. in Computer Network Technology, I never fully understood how the different AWS services worked together until completing this project.

One of my biggest takeaways was understanding that cloud computing is not simply one large computer. Instead, services such as Amazon S3, Amazon EC2, and Amazon VPC each perform a specific role while working together to support an application.
